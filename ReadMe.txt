Instructions to Run the Project:

1. Compilation:
   To compile the project, run the following command in the terminal:
   $ make

   This will compile both the Client and Server Java files.

2. Running the Server:
   To run the server, execute the following command:
   $ java Server [arguments]

   Replace [arguments] with any required arguments (e.g., DEBUG=1 or -p PORT_NUMBER or both).

3. Running the Client:
   To run the client, execute the following command:
   $ java Client [server-name] [file-name] [options]

   Replace [options] with the appropriate options for file transfer (e.g., -w, -s, -e, -p).
